# TextUtils

TextUtils is a powerful React-based text manipulation tool that offers a variety of features for analyzing and modifying text. It provides a user-friendly interface for tasks such as case conversion, word counting, and more.

## Features

- Text case conversion (uppercase, lowercase)
- Word and character counting
- Text-to-speech functionality
- Find and replace
- Remove extra spaces
- Dark and light mode
- Estimated reading time calculation
- Undo functionality
- Copy to clipboard

## Installation

To set up TextUtils locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-username/TextUtils-React-v1.git
   ```

2. Navigate to the project directory:
   ```
   cd TextUtils-React-v1
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to view the application.

## Usage

1. Enter or paste your text into the main text area.
2. Use the buttons below the text area to perform various text operations:
   - Click "Convert to Uppercase" to change all text to uppercase.
   - Click "Convert to Lowercase" to change all text to lowercase.
   - Use "Find & Replace" to search for specific text and replace it.
   - Click "Remove Extra Spaces" to eliminate unnecessary spaces.
   - Use "Text to Speech" to have the text read aloud.
3. The word count, character count, and estimated reading time will update automatically as you modify the text.
4. Use the theme toggle buttons in the navbar to switch between light, dark, and purple modes.
