# TextUtils

TextUtils is a powerful and user-friendly text manipulation tool built with React and TypeScript. It offers a variety of text processing features to enhance your writing and editing experience.

## Features

- **Case Conversion**: Easily convert text to uppercase or lowercase.
- **Text-to-Speech**: Listen to your text with built-in text-to-speech functionality.
- **Undo Operation**: Revert changes with a simple undo feature.
- **Text Clearing**: Clear the entire text area with one click.
- **Text Replacement**: Find and replace specific words or phrases.
- **Copy to Clipboard**: Quickly copy processed text to your clipboard.
- **Extra Space Removal**: Automatically remove unnecessary spaces from your text.
- **Dark and Light Mode**: Toggle between dark and light themes for comfortable viewing.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technologies Used

- React
- TypeScript
- React Router
- Bootstrap 5.1.0

## Project Structure

The project follows a standard React application structure:

- `src/`: Contains the source code of the application
  - `components/`: React components (Navbar, TextForm, About, Alert)
  - `App.tsx`: Main application component
  - `index.tsx`: Entry point of the React application
- `public/`: Static assets and HTML template
- `tests/`: Unit and component tests

## Installation and Setup

To run TextUtils locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-username/TextUtils.git
   cd TextUtils
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open your browser and visit `http://localhost:3000` to view the application.

## Usage

1. Enter or paste your text into the main text area.
2. Use the buttons above the text area to apply various text manipulations.
3. View the text summary below the text area for word count, character count, and estimated reading time.
4. Toggle between light, dark, and purple modes using the buttons in the navigation bar.
