# TextUtils

TextUtils is a React-based web application that provides various text manipulation utilities. It offers a user-friendly interface for transforming and analyzing text, with features like case conversion, text-to-speech, and more.

## Features

- Text transformation (uppercase, lowercase)
- Text-to-speech functionality
- Undo operation
- Text clearing
- Word and character replacement
- Text copying to clipboard
- Whitespace removal
- Word and character count
- Reading time estimation
- Theme toggling (Light, Dark, and Purple modes)
- Responsive design with Bootstrap

## Technologies Used

- React
- TypeScript
- React Router for navigation
- Bootstrap for styling
- Web Speech API for text-to-speech functionality

## Installation

To set up TextUtils locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-username/TextUtils.git
   ```

2. Navigate to the project directory:
   ```
   cd TextUtils
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to view the application.

## Usage

1. **Text Manipulation:**
   - Enter or paste your text in the main text area.
   - Use the buttons below to perform various operations on the text.

2. **Theme Toggle:**
   - Click on the theme buttons in the navbar to switch between Light, Dark, and Purple modes.

3. **Text Summary:**
   - View the word count, character count, and estimated reading time below the text area.

4. **About Section:**
   - Navigate to the About page to learn more about the application's features.

## Contributing

Contributions to TextUtils are welcome! Please feel free to submit a Pull Request.
